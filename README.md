# ПланПомещений

Конструктор помещений и расчет стоимости ремонта в Симферополе

## Описание

Веб-приложение для проектирования планировки помещений и автоматического расчета стоимости отделочных работ.

## Функциональность

- Создание плана помещения с комнатами, окнами и дверями
- Настройка параметров элементов (размеры, отделка)
- Автоматический расчет стоимости работ
- Формирование детальной сметы в виде кассового чека
- Отправка сметы в Telegram и WhatsApp
- Печать сметы
